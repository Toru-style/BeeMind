1. ハードスキル（技術・ツール）
Excel歴18年：社会人から一貫して使用。パターン化・見栄え・関数・ピボット・VBAまで習熟。
スプレッドシート＋GAS：スプレッドシートを簡易DBとして活用し、GASで自動化。
GitHub Pages実装経験：HTML生成→GitHub初Push→Pages公開まで。
開発経験（SE時代）：COBOL・VB（基幹系）を使用。6年半の業務経験あり。主にPM系業務。
文系出身だがITは独学吸収：実践中心。ロジックより検証重視。

2. 思考スタイル・習慣
選択と削ぎ落としの思考：「やること」と「やらないこと」の明確化。無駄が嫌い。
「シンプル追加型」思考：必要になったら加える、不要なものは持たない。
形式・構造重視：綺麗なアウトライン、定型化、反復性のある仕組みを好む。
検知力・特定力に優れる：「検証の鬼」。論理より感覚で問題を察知し、早期発見・解決。
記憶と整理欲求の強さ：頭の中で構造が出来上がっている状態が多く、記録はその反映。
アウトプット中心の記録型：ただの思考ではなく、記録と共有を前提に考えている。

3. 行動習慣・スタンス
スピード重視：資料作成・判断・修正・対応が早い。
「長時間やるのが嫌」→先に片付けて後で楽する型
再現性重視の運用：変更点のみ記録し、差分を追跡。GitHubも基本は必要時だけPush。
リビジョン管理志向：Excelや資料は定期コピー・バージョン保存が習慣。
プライベートと仕事の明確な分離：会社には残業ゼロ文化を定着。

4. 個人的特性
英語・外国語：海外生活2ヶ月で英語を話せるように。2言語最低レベルのコミュニケーション、聴き取りが出来るように。文法ではなく実用・実践型。
「感覚と勢い」タイプ：周囲には慎重・真面目に見られがちだが、実際は直感的・行動先行。
勘・流れ・パターン：偶然や運も含めた大局観を大事にする傾向あり。
「鬼シリーズ」：Excel、テスト、スピード、管理力、直感など、周囲から“鬼”と称されることも。
話すスピードが速く、周囲がついてこれないのが悩み

5. 使うツール（現役）
主力：Excel（スプレッドシート）
開発支援系：GAS、VBA
思考補助：ChatGPT（Bee）、Gemini（実験中）、iPhoneメモ、Google Drive系も一部

🧩 まとめの一言：
Tetu_stackは、「才能」ではなく「積層」の記録。
やってきたこと、身についたこと、考え方の癖もすべて、「何かをやり続けた結果」に過ぎない。
その中には、偶然や失敗もあるが、それすら重ねてきたことが、今のスタイルを支えている。
